# API Explorer

A developer tool for testing and exploring REST APIs. This app allows users to build and send HTTP requests, inspect reponses, and quickly replay previous requests.

## Live Demo

https://api-explorer-dashboard-rho.vercel.app/

## Features

- Send GET, POST, PUT, PATCH and DELETE requests
- Add custom request headers
- Send JSON request bodies
- View formatted JSON API responses
- View status codes and request timing
- Request hisotry panel to quickly replay previous requests
- Copy response JSON to clipboard
- Clean developer-focused UI

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Axios
- Vercel (deployment)

## Installation

Clone the repository:

> git clone https://github.com/bsilcox1990/api-explorer-dashboard.git
> cd api-explorer-dashboard

Install dependencies:

> npm install

Run the development server:

> npm run dev
